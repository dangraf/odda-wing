# odda-wing
Odens and Daniels flying wing project.
We are aiming to create a vtol rc flying wing that are fully autonomous.


steps:
1. Create a foam-cutter
2. Order parts
3. Assemble the wing
4. Make adjustments to optimize manual flight.
5. make it autonomous
